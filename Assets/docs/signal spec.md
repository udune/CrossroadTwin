# signal-spec.md — 신호 체계 최종 명세

> 적용 교차로: 우사단로 / 보광로 3거리 (한국폴리텍대학 인근)
> 사이클: **90초**
> 신호기 수: **6개** (보행자 3 + 차량 3)
> 최종 확정일: 2025

---

## 1. 교차로 구조

```
         12시 (보광로, 북쪽)
          │  좌회전 전용
          │
10시 ─────┼───────── 5시
(우사단로)  │         (우사단로)
  서쪽     │          동쪽
```

### 방향별 특징

| 방향 | 도로명 | 차로 | 허용 방향 |
|------|--------|------|-----------|
| 10시 | 우사단로 | 2차로 | 직진 + 좌회전 |
| 12시 | 보광로 | 1차로 | 좌회전 전용 |
| 5시 | 우사단로 | 2차로 | 직진만 |

## 4. 좌회전 신호 규칙

```
⚠️ 중요:
좌회전 화살표가 켜질 때 적색등도 반드시 동시에 점등

이유:
  - 한국 도로교통법상 좌회전 = 적색등 + 좌회전 화살표 동시
  - 운전자에게 "직진 불가, 좌회전만 가능" 명시

Unity 구현:
  Y0C8 (10→ 좌회전) ON 시 → Y0C6 (10→ 적색)도 ON
  Y0CC (12→5 좌회전) ON 시 → Y0CA (12→5 적색)도 ON
```

---

## 5. 황색 신호 규칙

```
황색 = 페이즈 종료 신호 (각 페이즈 마지막 3초)

페이즈 A 끝: 19~21초 → 10→ 황색
페이즈 B 끝: 46~48초 → 12→5 황색
페이즈 C 끝: 88~90초 → 10→ 황색 + 5→ 황색 동시

⚠️ 황색 중에는 해당 차량 신호만 황색
   다른 신호들은 적색 유지
```

## 6. 인터락 (충돌 방지 규칙)

동시에 켜지면 안 되는 신호 조합:

| 규칙 | 설명 |
|------|------|
| 10→ 좌회전 ↔ 12→5 좌회전 | 두 좌회전 동시 금지 |
| 10→ 좌회전 ↔ 5→ 직진 | 좌회전 중 직진 차량 진입 금지 |
| 12→5 좌회전 ↔ 10→ 직진 | 좌회전 중 직진 차량 진입 금지 |
| 차량 녹색 ↔ 해당 방향 횡단보도 녹색 | 차량-보행자 동시 금지 |

## 7. 특수 기능 신호 상태

### 보행자 모드 (차량 3곳 미감지 10초)

```
모든 차량 신호: 적색
10시 횡단보도: 녹색
12시 횡단보도: 녹색
5시 횡단보도: 녹색
D0 카운터: 정지
```

### 긴급 모드 — 황색 단계 (5초)

```
모든 차량 신호: 황색
모든 횡단보도: 적색
D0 카운터: 정지
```

### 긴급 모드 — 적색 깜빡임 단계 (10초)

```
모든 신호: 적색 깜빡임 (0.5초 주기, SM412)
D0 카운터: 정지
```

### 노약자 연장

```
현재 신호 상태 유지
D0 카운터 정지 (10초)
10초 후 자동 재개
```
