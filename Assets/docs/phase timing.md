# phase-timing.md — 페이즈 타이밍 명세

> 전체 사이클: **90초**
> 페이즈 순서: **A → B → C → A → ...**
> 최종 확정일: 2025

---

## 1. 페이즈 개요

| 페이즈 | 시간 범위 | 길이 | 핵심 동작 |
|--------|-----------|------|-----------|
| **페이즈 A** | 1~21초 | 21초 | 10→ 좌회전 + 5시 횡단 녹색 |
| **페이즈 B** | 22~48초 | 27초 | 12→5 좌회전 + 10시 횡단 녹색 |
| **페이즈 C** | 49~90초 | 42초 | 양방향 직진 + 12시 횡단 녹색 |

---

## 2. 페이즈 A 상세 (1~21초)

```
1~18초:  녹색 구간 (18초)
  - 10→ 차량: 좌회전 녹색화살표 + 적색등 동시 점등
  - 5시 횡단보도: 보행자 녹색
    - 1~9초: 녹색 점등
    - 10~18초: 녹색 깜빡임

19~21초: 황색 구간 (3초)
  - 10→ 차량: 황색
  - 5시 횡단보도: 적색 전환
```

### 페이즈 A 신호 상태표

| 신호기 | 1~18초 | 19~21초 |
|--------|--------|---------|
| 10시 횡단보도 | 적색 | 적색 |
| 12시 횡단보도 | 적색 | 적색 |
| 5시 횡단보도 | **녹색** (깜빡임 10~18초) | 적색 |
| 10→ 차량 | **좌회전** (적색등 동시) | **황색** |
| 12→5 좌회전 | 적색 | 적색 |
| 5→ 차량 | 적색 | 적색 |

---

## 3. 페이즈 B 상세 (22~48초)

```
22~45초: 녹색 구간 (24초)
  - 12→5 차량: 좌회전 녹색화살표 + 적색등 동시 점등
  - 10시 횡단보도: 보행자 녹색
    - 22~36초: 녹색 점등
    - 37~45초: 녹색 깜빡임

46~48초: 황색 구간 (3초)
  - 12→5 차량: 황색
  - 10시 횡단보도: 적색 전환
```

### 페이즈 B 신호 상태표

| 신호기 | 22~45초 | 46~48초 |
|--------|---------|---------|
| 10시 횡단보도 | **녹색** (깜빡임 37~45초) | 적색 |
| 12시 횡단보도 | 적색 | 적색 |
| 5시 횡단보도 | 적색 | 적색 |
| 10→ 차량 | 적색 | 적색 |
| 12→5 좌회전 | **좌회전** (적색등 동시) | **황색** |
| 5→ 차량 | 적색 | 적색 |

---

## 4. 페이즈 C 상세 (49~90초)

```
49~66초: 직진 + 12시 횡단 녹색 구간 (18초)
  - 10→ 차량: 직진 녹색
  - 5→ 차량: 직진 녹색
  - 12시 횡단보도: 보행자 녹색
    - 49~57초: 녹색 점등
    - 58~66초: 녹색 깜빡임

67~87초: 직진 지속 구간 (21초)
  - 10→ 차량: 직진 녹색 (지속)
  - 5→ 차량: 직진 녹색 (지속)
  - 12시 횡단보도: 적색 전환 (67초부터)

88~90초: 황색 구간 (3초)
  - 10→ 차량: 황색
  - 5→ 차량: 황색
```

### 페이즈 C 신호 상태표

| 신호기 | 49~66초 | 67~87초 | 88~90초 |
|--------|---------|---------|---------|
| 10시 횡단보도 | 적색 | 적색 | 적색 |
| 12시 횡단보도 | **녹색** (깜빡임 58~66초) | 적색 | 적색 |
| 5시 횡단보도 | 적색 | 적색 | 적색 |
| 10→ 차량 | **직진 녹색** | **직진 녹색** | **황색** |
| 12→5 좌회전 | 적색 | 적색 | 적색 |
| 5→ 차량 | **직진 녹색** | **직진 녹색** | **황색** |

## 6. 보행자 신호 깜빡임 규칙

```
모든 보행자 신호:
  녹색 점등 = 총 녹색 시간의 절반
  녹색 깜빡임 = 총 녹색 시간의 절반

페이즈별:
  5시 횡단 (18초): 점등 9초 + 깜빡임 9초
  10시 횡단 (24초): 점등 15초 + 깜빡임 9초
  12시 횡단 (18초): 점등 9초 + 깜빡임 9초
```

## 7. D0 카운터와 페이즈 매핑

```
D0 값    페이즈   이벤트
1        A 시작   10→ 좌회전 ON, 5시 횡단 녹색 ON
10               5시 횡단 깜빡임 시작
19               황색 전환
22       B 시작   12→5 좌회전 ON, 10시 횡단 녹색 ON
37               10시 횡단 깜빡임 시작
46               황색 전환
49       C 시작   10→ 직진 ON, 5→ 직진 ON, 12시 횡단 녹색 ON
58               12시 횡단 깜빡임 시작
67               12시 횡단 적색 전환
88               황색 전환
90 → 1   A 시작   사이클 리셋
```

## 8. 특수 기능 적용 시 타이밍 변화

### 보행자 모드 (M200 ON)

```
D0 카운터 정지
3곳 횡단보도 동시 녹색
모든 차량 신호 적색
차량 감지 시 즉시 해제 → D0 재개
```

### 노약자 연장 (M203 ON)

```
D0 카운터 정지
현재 신호 상태 유지
T200 (10초) 완료 후 D0 재개

적용 시점:
  5시 횡단 노약자: D0 = 18 (페이즈 A 녹깜 끝)
  10시 횡단 노약자: D0 = 45 (페이즈 B 녹깜 끝)
  12시 횡단 노약자: D0 = 66 (페이즈 C 녹깜 끝)
```

### 조기 전환 (차량 미감지 10초)

```
페이즈 C 진행 중
  3곳 센서 모두 10초 미감지
    → D0를 88초로 강제 설정 (황색 구간 진입)
    → 황색 3초 후 페이즈 A 시작
    → 다음 사이클 정상 진행
```

### 긴급 모드 (M201 ON)

```
M2011 (황색 단계): 5초
  → 모든 차량 황색, 모든 횡단보도 적색

M2012 (적색 깜빡임): 10초
  → 모든 신호 적색 깜빡임 (SM412 사용)

자동 해제 → D0 재개
```
